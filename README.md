## プログラムを修正して貢献したい
プログラムを作りたいって！いいーじゃない！  
以下の手順でお願いね。

- このリポジトリをフォークしてね。
- フォークしたリポジトリに新しいブランチ作って。 (`git checkout -b improve-feature`)
- ソースコードを変更してコミットしてね。 (`git commit -am 'すげー機能作ったよ！'`)
- プッシュしよう！ (`git push origin improve-feature`)
- プルリクエストを作ろう！
